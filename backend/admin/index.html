Admin Console

Same-origin, memory-only admin surface for shared seats, invite delivery, recovery delivery, and audit review. This does not run calculator logic and it does not persist browser secrets.

Trust boundary Backend seats, sessions, invite links, and recovery links only.
Storage posture API keys, seat codes, and session tokens stay in browser memory only.
Recommended ops mode Issue a short-lived seat session, then use that session for admin actions.
