# Plant Engineering Surface Guide

## Purpose

This guide explains the `Current working surface` block now shown in plant-capable jobs.

It exists to answer one practical question:

- which workflow should lead the next decision now that the site is no longer a simple one-board quote?

## The Three Surface States

### `System Design Surface`

Use this when the project is still a normal captive-site system.

Meaning:

- the standard design result is still the main working surface
- the plant-only blocks are secondary or not needed
- the quote can stay inside the normal installer/proposal workflow

### `Plant Engineering Surface`

Use this when the project is still inside the product boundary, but feeder, board, source, and operating handoff now matter as much as the main sizing summary.

Meaning:

- keep using the same product
- treat plant outputs as part of the main installer handoff
- carry feeder, board/source, breaker/cable, and dispatch notes together

### `Utility / Mini-Grid Engineering Surface`

Use this when the project has crossed beyond the normal captive-site lane.

Meaning:

- the current app is still useful for early scoping
- final work must move into a separate utility / mini-grid engineering workflow
- interconnection, protection, dispatch, and commissioning depth now exceed the normal site-design lane

## Why This Matters

Without this block, complex jobs can drift in two bad directions:

- the user keeps reading everything like a simple site design
- the app gets blamed for work it never claimed to finish

The working-surface block fixes that by making the lane explicit before the quote or handoff is oversold.

## How To Use It

1. Read `Current working surface` before the deeper plant blocks.
2. If it says `System Design Surface`, stay in the normal design/proposal lane.
3. If it says `Plant Engineering Surface`, keep the plant blocks open and carry them into handoff.
4. If it says `Utility / Mini-Grid Engineering Surface`, stop treating the current output as a final engineering basis.

## What It Does Not Claim

This block does not mean the app already completed:

- feeder studies
- selectivity studies
- formal dispatch studies
- interconnection studies
- stamped engineering
